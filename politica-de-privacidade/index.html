Política de privacidade

Este site tem caráter informativo e utiliza os dados estritamente necessários para contato, navegação e segurança básica da experiência.

Quais dados podem aparecer no uso do site

  • Dados de navegação anônimos, como páginas visitadas e cliques em links.
  • Informações que você decidir enviar no contato inicial por WhatsApp.
  • Dados técnicos básicos gerados pelo navegador e pela hospedagem para segurança e funcionamento.

Como esses dados são usados

Os dados são utilizados para viabilizar a navegação, responder mensagens e compreender de forma agregada quais páginas ajudam mais pessoas a encontrar o atendimento.

Não há comercialização de dados pessoais. O uso de qualquer informação segue os princípios de necessidade, finalidade e cuidado compatíveis com a natureza do serviço oferecido.

Seus direitos

Você pode solicitar esclarecimentos sobre dados pessoais eventualmente compartilhados, pedir correção de informações e solicitar exclusão quando isso for aplicável.

Para qualquer demanda desse tipo, o canal mais rápido é o contato direto pelo WhatsApp informado no site.
